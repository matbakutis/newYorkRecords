# newYorkRecords

## General Assembly SECA
A user interface for the New York Records API.
Created by Mat Bakutis for project 3 in General Assembly's SECA program.

## How to Use
```
$ git clone
$ cd back-end-users
$ docker-compose up
```
## How to Test
```
$ git clone
$ cd back-end-users
$ ./gradlew test
```
