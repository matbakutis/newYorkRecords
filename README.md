# newYorkRecords

## General Assembly SECA
A user interface and forum for the New York Records API.
Created by Mat Bakutis for project 34in General Assembly's SECA program.

## How to Use
```
$ git clone
$ cd back-end-users
$ docker-compose up
```
## How to Test
```
$ git clone
$ cd back-end-users
$ ./gradlew allTests
```
